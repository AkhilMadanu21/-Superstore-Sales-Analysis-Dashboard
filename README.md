## 🔍 Overview
This project presents a comprehensive **Sales Dashboard** built using **Power BI** to analyze Superstore sales, profits, and orders across **2022–2025**. It also includes supporting files in CSV, PDF, and PowerPoint formats for presentation and reporting purposes.

- **Total Sales:** $187.49K

- **Total Orders:** 365  
- **Average Discount:** 15%  
- **Average Quantity:** 5.46  

---

## 📊 Dashboard Highlights

### 🟢 Sales Overview
- **Technology** is the top-selling category.
- **South** leads in regional sales (27.77%).
- **Consumer** segment drives the highest revenue (37.42%).
- Monthly sales trends tracked over 4 years.

### 🔵 Profit Analysis
- Highest profitability from **Corporate segment** (40.38%).
- Losses detected in low-margin products like **Bankers Box**.
- Need for better discount control in Office Supplies.

### 🟠 Order Insights
- **Furniture** has the highest order volume.
- Repeated orders from high-utility items like **Xerox Paper**.
- South again leads in total order share (27.93%).

---

## 🧠 Strategic Recommendations
- Target high-profit segments like **Corporate**.
- Reassess pricing and discounts in low-margin items.
- Improve inventory for high-frequency SKUs.
- Focus marketing efforts in **Central & East** to boost reach.
